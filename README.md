## 2. 코루틴의 특징
  - 기본옵션으로 협력형이며 병행형으로 동작하지 않는다 (옵션에따라 여러개의 스레드로 분할될수 있음)
  - 스케줄러가 실행시점을 결정하는 것이 아닌 프로그래머나 이벤트에 의해 실행 및 지연, 재개 시점이 결정된다
  - 비선점형 멀티태스크방식 , 동시성을 제공한다
  - 독립적 스택을 가질 수도 있으나 일반적으로 스택을 다시 생성하지 않는다.
